# java-cli-buildr-sieve-atkins

## Description
Given a number N, print all prime numbers smaller than N.

## Tech stack
- java
- buildr

## Docker stack
- vanto/apache-buildr:latest-jruby-jdk8

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
